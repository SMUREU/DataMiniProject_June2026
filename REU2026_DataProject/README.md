# REU2026_DataProject 

**Author:** Charlotte Lin

## Project Overview

In this project, I explored potential correlations between country-level factors and infant sleep duration to better understand why infant sleep times vary across countries.

Because there was no one clean dataset on infant sleep times, I created my own dataset by collecting and combining data from multiple research papers on infant sleep time and pulled additional data from World Bank Open Data.

The data_visual.py script uses this dataset to generate visualizations that examine several possible correlations between infant sleep duration and country-specific variables.
